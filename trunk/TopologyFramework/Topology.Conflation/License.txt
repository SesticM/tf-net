Topology Framework .NET (TF.NET) is a collection of .NET classes that
implement the fundamental operations required to perform various kinds
of geo-spatial conflation processes. This collection of classes is a
port (with some additions and modifications) of JCS Conflation Suite
(see next license for more informations).
